Rebuild dataset metadata

Fully rebuilds the dataset's static metadata database by scanning every data type's parquet files (records, embeddings, fulltexts). Use this after a bulk load or when the metadata looks stale or out of sync with the records.

⚠️ This writes to the dataset: it clears the append deltas and overwrites the canonical metadata.duckdb. On a large dataset it can take several minutes.
